## 🎤 Work Story

- ### 프로젝트 투입 배경

  기존 Kara App 운영 웹 퍼블리셔가 다른 팀의 프로젝트로 이동 되는 이유로 교체 인력으로 제가 투입 되었습니다.
  원래는 종료일이 12월 24일 계약으로 제가 투입 하였지만 여러가지 이유로 10월 22일에 철수 하였습니다.

  <small>철수 사유는 인터뷰 때 말씀 드리겠습니다.</small>

- ### 프로젝트 이슈 및 대응 사항

  1. 기존 웹퍼블리셔가 해결 하지 _못했던 이슈_ 들을 _대응 조치 완료_ 하였습니다.

     > _아이폰에서만_ 특정 페이지의 *팝업 위치가 하단 고정이 되지 않은 이슈*가 있었는데 확인 해보니 일부 페이지의 모달의 css가 _safe-area-inset-bottom_ 속성이 _문제_ 였는데 이 부분을 현재 css 상황에 맞게 다시 조치 하여 _해결_ 하였습니다. <br/><small>자세한 내용은 인터뷰 때 말씀 드리겠습니다.</small>

     > 달력 테이블에서 세로 정렬이 되지 않는 이슈, ionic shadow dom 관련 이슈들을 대응 하였습니다.

  2. _운동_ 메뉴가 추가 되어 관련 _신규 페이지 퍼블리싱_, _Kare App 메인 퍼블리싱_, *브랜드 반응형 사이트 퍼블리싱*을 완료 하고 철수 하였습니다.
